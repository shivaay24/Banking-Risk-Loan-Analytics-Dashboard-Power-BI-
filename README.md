## Banking Risk & Loan Analytics Dashboard (Power BI)

This project focuses on analyzing banking loan data to identify customer risk patterns and improve lending decisions using data analytics.

### Project Overview

The dashboard analyzes 1000+ banking records including customer information, loan data, and credit score distribution to provide insights into loan performance and potential risks.

### Key Dashboard Features

* Total loan amount and default rate tracking
* Credit score segmentation (High Risk / Medium Risk / Low Risk)
* Loan analysis by city, occupation, tenure, and loan type
* Interactive filters for dynamic exploration

### Key Insights

* Customers with lower credit scores show higher default risk
* Loan distribution varies significantly across cities and occupations
* Risk segmentation helps banks improve credit decision strategies

### Tools Used

* Power BI
* DAX (Data Analysis Expressions)
* Data Visualization

### Purpose

To demonstrate how data analytics and visualization can support **risk assessment and strategic decision-making in the banking sector**.
